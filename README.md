# 💳 Digital Payment Fraud Analysis Dashboard

An interactive **Digital Payment Analytics Dashboard** built using **Python**, **Streamlit**, **Plotly**, and **SQLite** to analyze India's UPI transaction ecosystem using the official **PhonePe Pulse** dataset.

The dashboard provides comprehensive transaction insights through interactive visualizations, key performance indicators (KPIs), and filtering options, helping users understand digital payment trends across different time periods.

> **Note:** The current version focuses on transaction analytics and business insights. Machine Learning-based fraud detection is planned as a future enhancement.

# 📖 Project Overview

India has witnessed tremendous growth in digital payments through the Unified Payments Interface (UPI). Understanding transaction patterns can provide valuable insights into user behavior, payment trends, and the evolution of the digital economy.

This project analyzes national-level UPI transaction data from the official **PhonePe Pulse** repository and presents the results through an interactive dashboard.

The dashboard enables users to:

- Explore UPI transaction trends
- Analyze transaction categories
- Compare quarterly growth
- Monitor transaction volumes
- Gain business insights using interactive visualizations
